Este blog está realizado con Jekyll, un generador de páginas web estáticas desarrollado en Ruby.
Sobre la configuración por defecto he realizado algunas modificaciones mínimas a los estilos e incorporado algunos plugins y fragmentos HTML.

Por si os interesan, estos son los añadidos:

1. Fragmentos HTML

  • Disqus: para gestionar los comentarios
  • Google Analytics: para llorar un poco con la presencia del blog en internet.
  • Compartir enlaces: para incluir el botón de Karmacracy y el de Twitter.

2. Plugins

  • i18n: permite traducir algunas variables de Jekyll, como las fecha de publicación.
  • karmacracy: este es de cosecha propia. Acorta las url´s de los artículos para el tweet de publicación de nuevas entradas.
  • slideshare: para incrustar presentaciones de Slideshare en los artículos.
  • tweet_tag: para incrustar tweets en los artículos.
  • vimeo: para incrustar videos de vimeo en los artículos.

3. Octopress 3.0

Octopress ha dejado de ser un generador de blogs basado en Jekyll y ahora es un conjunto de plugins que facilitan el mantenimiento de un blog hecho con Jekyll. Estos son los que se usan en este blog:

  • Octopress 3.0: incluye ayudas para el mantenimiento del blog (creación de entradas, borradores) pero, sobre todo, simplifica enormemente el proceso de publicación en Github Pages.
  • Octopress Solarized: modifica el css del código fuente encastrado dentro de los artículos.
  • Octopress Quote Tag: facilita la creación de citas en los artículos.
  • Octopress Blockquote: aunque estoy empezando a usar más la etiqueta highligth de Jekyll, todavía quedan unos cuantos artículos de la época en la que este blog estaba hecho en Octopress 2.0
  • Octopress Linkblog: simplifica la creación de artículos en los que comento otros artículos o web.
