Comandos útiles

chown permite cambiar el propietario de un fichero. Esto es util cuando tienes un script dentro de la carpeta de usuario y quieres que se ejecute en un cron.

sudo chown root:root /home/pi/scripts/publicar.sh

Para listar el contenido del mail:

tail /var/mail/pi

Para ver el log del sistema:

tail /var/log/syslog
